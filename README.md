# Khando — MVP կայք (Next.js + Vercel)

Այս նախագիծը պատրաստված է ըստ ձեր `կայք.docx` ֆայլում նկարագրված կառուցվածքի՝
- Վերին մենյու (խոշոր, ընթեռնելի)
- Գլխավոր էջ՝ մեծ հերո նկար + արագ բլոկներ + «Մեր գյուղը» + մեդիա շերտ + մասնակցության կոչ
- Բաժիններ՝ Մեր գյուղը, Գյուղի կյանքը, Համայնք, Դպրոց, Մշակույթ, Գյուղատնտեսություն, Սփյուռք, Մարդիկ, Մեդիա, Կարծիքներ/Կապ

## Տեղադրում (տեղային)
```bash
npm install
npm run dev
```
Բացեք՝ http://localhost:3000

## Deploy Vercel (առաջարկվող)
1) GitHub-ում ստեղծեք նոր repo և այս ֆայլերը push արեք
2) Vercel → **New Project** → ընտրեք repo → Deploy  
3) Դոմենը կարող եք կապել (օր. `khando.am`)

## Ինչ փոխել առաջինը
- `public/hero.jpg` — փոխեք իրական գյուղի լայն լուսանկարով
- `public/gallery/*.jpg` — փոխեք իրական նկարներով
- `data/content.ts` — լրացրեք նորություններ/միջոցառումներ/արագ բլոկներ
- `components/footer.tsx` — կոնտակտներ և սոցիալական հղումներ

## Մասնակցություն (ֆորմա)
`/qarcikner-kap#participate` էջում կա ֆորմա, որը ուղարկում է POST դեպի `/api/participate`.

Ներկայումս դա միայն **console.log** է անում։  
Ավելի լավ տարբերակներ՝
- Vercel Postgres / Supabase՝ պահելու համար
- Email ուղարկում (Resend / SMTP)
- Admin էջ՝ հաստատելու և հրապարակելու համար

Եթե ցանկանում եք՝ ես կարող եմ ավելացնել տվյալների պահեստ (Supabase կամ Vercel Postgres) և admin մուտք։
